# Analyzing-Australian-Open-Matches-2008-2022-
A statistical exploration of player performance, ranking differences, and upset frequencies in professional tennis. This project uses R and Quarto to visualize and summarize trends in the Australian Open dataset spanning 15 years.

## Overview

Using R libraries like `dplyr`, `tidyr`, and `tibble`, this project:
- Calculates total matches and unique player counts
- Examines win percentages by player
- Categorizes match outcomes by ranking differences
- Identifies matches with the largest rank upsets
- Computes upset proportions across rank difference categories
